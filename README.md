#To Do App#

The structures used in this application are: 
  - SQLite
  - Room
  - ViewModel 
  - Dagger-Hilt (Dependency Injection)
  - Repo
  - Fragment
  - Navigation 
  - Adapter

The operations performed in the application include: 
  - Adding a new to-do 
  - Deleting a to-do 
  - Updating a to-do 
  - Searching for a to-do
